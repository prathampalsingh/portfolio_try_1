    FEB - JUL 2025: LenDenClub Android APP Development

    Kotlin, Android Studio, Jetpack Compose, XML

    I contributed to the development of the LenDenClub Android application, a high-traffic finance platform with a user base exceeding 85 lakh users. The app follows a clean MVVM architecture and was originally built using an XML-driven UI. My work involved delivering several high-impact features, architectural improvements, and performance optimizations.

    Key Contributions:
    • Led the migration of the complex, performance-heavy XML-based Dashboard to Jetpack Compose, resulting in an 80% faster rendering time, smoother UI interactions, and a significantly cleaner, more modular codebase ( Spot Award – July 2025).
    • Refactored and redesigned the Portfolio module, introducing responsive UI components, optimized recomposition logic, and enhanced data visualization for improved portfolio insights and user engagement.
    • Overhauled the user onboarding flow by streamlining the navigation graph, reducing CTA friction, and improving form validation UX, increasing onboarding completion rates by 22%.
    • Implemented full DeepLink support across the app, enabling seamless navigation, personalized user flows, and integration with marketing campaigns and push notifications.
    • Improved application stability by identifying and resolving lifecycle-related crashes, memory leaks, and redundant observers.
